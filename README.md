# 🚀 Programação & QA: Semana 01

Repositório dedicado aos exercícios práticos da primeira semana de estudos em tecnologia, focando em lógica de programação com **Go** e **JavaScript**, com ênfase em **Qualidade de Software (QA)**.

## 🛠️ Tecnologias Utilizadas
* **Linguagens:** Go (Golang) & JavaScript (Node.js)
* **Testes:** Jest (JavaScript Unit Testing)
* **Qualidade:** ESLint & JSDoc (Documentação)

---

## 📂 Estrutura do Projeto

### 🐹 [GO](./GO)
Exercícios focados em sintaxe básica, tipos de dados e lógica inicial.
* `main.go`: Hello World
* `even_odd.go`: Verificador de Par/Ímpar
* `fizzbuzz.go`: Desafio FizzBuzz
* `tempconv.go`: Conversor de Temperatura
* `guessing.go`: Jogo de Adivinhação

### 🟨 [JavaScript](./javascript)
Exercícios avançados com foco em **Testes Automatizados**.
* **Testes Unitários:** Cada exercício possui um arquivo `.test.js` correspondente.
* **Cobertura:** 100% de cobertura de lógica com Jest.
* **Documentação:** Uso de JSDoc para tipagem e clareza de funções.

---

## 🧪 Como rodar os testes (QA)

Para validar a qualidade do código JavaScript, utilize o Jest:

1. Entre na pasta: `cd javascript`
2. Instale as dependências: `npm install`
3. Execute os testes: `npx jest`
